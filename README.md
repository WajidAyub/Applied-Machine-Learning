# Applied Machine Learning Portfolio

Welcome to my Applied Machine Learning Portfolio! This repository contains a curated collection of my most impactful machine learning projects, demonstrating my ability to apply fundamental algorithms to real-world datasets across various domains including healthcare, telecommunications, and astrophysics.

## 🚀 Projects Overview

| Project | Domain | Key Algorithms / Techniques | Highlights |
|---------|--------|-----------------------------|------------|
| **[01. Titanic Survival Analysis](./01_Titanic_Survival_Analysis/)** | Demographics | Logistic Regression, EDA | Comprehensive Exploratory Data Analysis, Feature Importance Analysis, ROC Curve Visualization |
| **[02. Telecom Churn Prediction](./02_Telecom_Churn_Prediction/)** | Business | Ridge Regression, Feature Scaling | Predicting customer churn, applying regularization (L2) to prevent model overfitting |
| **[03. Heart Disease Prediction](./03_Heart_Disease_Prediction/)** | Healthcare | Logistic Regression | Building a predictive diagnostic model using structured medical data |
| **[04. Pulsar Star Classification](./04_Pulsar_Star_Classification/)** | Astrophysics | Classification | Handling and evaluating models on complex, imbalanced astronomical datasets |
| **[05. Decision Tree From Scratch](./05_Decision_Tree_From_Scratch/)** | Core ML | ID3 Algorithm, Information Gain | **Implemented ID3 completely from scratch**, calculating Entropy and Information Gain manually |
| **[06. Breast Cancer Classification](./06_Breast_Cancer_Classification/)** | Healthcare | Random Forest vs Logistic Regression | Model comparison, advanced evaluation metrics (Precision, Recall, F1-Score, ROC-AUC) |

## 🛠️ Technologies & Skills
* **Languages:** Python
* **Libraries:** `pandas`, `numpy`, `scikit-learn`, `matplotlib`
* **Core Skills:** Exploratory Data Analysis (EDA), Data Preprocessing, Feature Engineering, Classification, Regression, Model Evaluation, Implementing Algorithms from Scratch.

## ⚙️ How to Run
To run any of the notebooks or scripts locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/WajidAyub/Applied-Machine-Learning.git
   ```
2. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Navigate to the project folder and run the Python script or open the Jupyter Notebook.
